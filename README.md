## 프로젝트 소개 🧞

코로나19로 인해 비대면 수업이 확대되면서 **청각 장애를 가진 학생들의 학습권**은 보장 받지 못하고 있습니다.<br/>
<br/>
강의 화면에 교수자의 입모양이 나오지 않고, 기본적인 자막 또한 보장되지 않기 때문입니다. <br/>
청각 장애 학생들이 지원 방안 마련을 촉구하는 국민 청원을 올리기도 하였으나 마땅한 대책이 마련되지 않는 실정입니다.<br/>
[자세히 알아보기 (온라인 강의에서 소외되는 청각장애인 학생들 / 스브스뉴스) ](https://www.youtube.com/watch?v=iHgDgymoGNs)<br/>
<br/>
이러한 어려움을 해결하기 위하여 **청각 장애인을 위한 실시간 자동 강의록** 을 개발하게 되었습니다.
